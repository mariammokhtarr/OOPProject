Heroes
-Warrior: A hero who attacks with a sword
-Mage: A hero who attacks using spells
-Archer: A hero who attacks with arrows


Each hero has Health and power
Heroes attack each others 
When hero’s health become zero the hero is dead
Hero can heal using heal() method


Use Abstract Class (BaseHero) and other classes inherited it
Implements (Encapsulation) for private health and power fields.
Demonstrates (Polymorphism) with Attack methods for all heroes 
Stores heroes in an Array and use if /else to show if hero is alive or dead
